# Smalltalk 插件

闲聊插件，提供轻松愉快的对话和趣味回复功能。

## 功能介绍

Smalltalk 插件提供各种有趣的闲聊命令，包括讲笑话、问候、鼓励、随机回复等轻松愉快的互动功能。

## 使用方法

### 基本命令

```
/joke                   # 讲一个笑话
/hi                     # 打招呼
/encourage              # 获得鼓励
/random                 # 随机回复
/time                   # 当前时间
/help smalltalk         # 显示帮助信息
```

### 示例

```
/joke                   # 听一个笑话
/hi                     # 向机器人打招呼
/encourage              # 需要一些鼓励
/random                 # 看看机器人会说什么
/time                   # 查看当前时间
```

## 功能特性

- 多样化的互动命令
- 随机回复增加趣味性
- 支持自定义回复内容
- 轻量级无需配置
- 支持 emoji 表情

## 适用场景

- 活跃群聊氛围
- 缓解紧张情绪
- 日常轻松互动
- 测试机器人响应

## 扩展方式

可以通过编辑插件代码添加更多闲聊内容：

1. 添加新的笑话到笑话库
2. 增加鼓励语句
3. 扩展问候语
4. 添加更多随机回复

## 注意事项

- 内容仅供娱乐，不涉及敏感话题
- 回复内容随机，每次可能不同
- 适合在非正式场合使用
