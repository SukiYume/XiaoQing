# Color 颜色查询插件

## 功能简介

本插件是一个功能强大的颜色查询和转换工具，提供：
- **8000+ 中国传统色彩**：查询中国传统颜色的 RGB、HEX、CMYK 值
- **恒星光谱颜色**：查询不同光谱型恒星的颜色
- **颜色格式转换**：支持 RGB、HEX、CMYK 之间的转换
- **自定义颜色库**：添加和管理自己的颜色
- **颜色可视化**：生成颜色示例图片

## 主要特性

- 🎨 **丰富的颜色库**：内置8000+中国传统色彩数据
- 🔄 **格式转换**：RGB ↔ HEX ↔ CMYK 互转
- 🌟 **恒星颜色**：查询恒星光谱型对应的颜色
- 🖼️ **可视化**：自动生成颜色示例图片
- 💾 **自定义库**：添加、删除自定义颜色
- 🔍 **智能搜索**：按名称、色系、颜色值搜索
- 🛡️ **健壮性**：完善的参数验证和错误处理
- 📝 **详细日志**：记录所有操作便于问题排查

## 使用方法

### 基础查询

#### 按名称查询

```
/颜色 -n 胭脂
/color -n 天青
/色彩 -n 竹青 -p    # -p 显示图片
```

**输出示例：**
```
name: 胭脂
RGB: [156, 28, 53]
hex: #9c1c35
CMYK: [26, 95, 82, 21]
```

#### 按 RGB 查询

```
/color -r 255,87,51
/颜色 -r 255 87 51     # 支持空格或逗号分隔
/color -r 255,87,51 -p # 显示图片
```

#### 按 HEX 查询

```
/color -x #FF5733
/color -x FF5733      # # 号可选
```

#### 按 CMYK 查询

```
/color -c 0,100,100,0
/color -c 0 100 100 0
```

### 色系搜索

```
/颜色 -a 红           # 搜索所有红色系
/color -a 蓝          # 搜索所有蓝色系
/色彩 -a 青           # 搜索所有青色系
```

**输出示例：**
```
朱红，绯红，胭脂，茜红，赤红，海棠红，樱桃红...
```

### 自定义颜色

#### 添加颜色

```
# 使用 RGB 值
/color -w 我的红 255 0 0

# 使用 HEX 值
/color -w 我的蓝 #0000FF
```

#### 删除颜色

```
/color -d 我的红
```

### 恒星光谱颜色

#### 查询特定光谱型

```
/color -s O5V        # 查询 O5V 型恒星颜色
/color -s M0V        # 查询 M0V 型恒星颜色
```

**输出示例：**
```
🌟 恒星光谱颜色

光谱型: O5V
HEX: #9BB0FF
RGB: [155, 176, 255]
```

#### 列出所有光谱型

```
/color -t            # 列出所有光谱型
/color -t O          # 列出 O 型相关光谱
/color -t M          # 列出 M 型相关光谱
```

### 帮助信息

```
/color -h
/color --help
/color -l
```

## 颜色格式说明

### RGB 格式
- 格式：`R,G,B` 或 `R G B`
- 范围：每个值 0-255
- 示例：`255,87,51` 或 `255 87 51`

### HEX 格式
- 格式：`#RRGGBB` 或 `RRGGBB`
- 支持简写：`#RGB`（自动扩展为 #RRGGBB）
- 示例：`#FF5733` 或 `FF5733`

### CMYK 格式
- 格式：`C,M,Y,K` 或 `C M Y K`
- 范围：每个值 0-100
- 示例：`0,100,100,0` 或 `0 100 100 0`

## 错误处理

插件包含完善的错误处理机制：

### 格式错误
```
❌ RGB 值必须在 0-255 范围内，第1个值为 300
❌ HEX 格式错误：无效的 HEX 颜色值: #GGGGGG
❌ CMYK 需要4个值，实际提供了3个
```

### 依赖缺失
```
❌ 图片生成失败：缺少 matplotlib 和 numpy 依赖
❌ 恒星颜色查询功能不可用
   需要安装 pandas 依赖：pip install pandas
```

### 数据未找到
```
❌ 中国色里没有收录「xxx」这个颜色哦。
❌ 没有找到光谱型「XXX」的恒星颜色
```

## 技术细节

### 颜色转换算法

#### RGB → CMYK
```
如果 RGB = (0, 0, 0): CMYK = (0, 0, 0, 100)
否则:
  C = (1 - R/255)
  M = (1 - G/255)
  Y = (1 - B/255)
  K = min(C, M, Y)
  C = (C - K) / (1 - K) * 100
  M = (M - K) / (1 - K) * 100
  Y = (Y - K) / (1 - K) * 100
  K = K * 100
```

#### RGB ↔ HEX
```
RGB → HEX: #%02X%02X%02X
HEX → RGB: 每两位十六进制转十进制
```

### 图片生成

### 搜索优化

- 支持部分匹配和关键词搜索

## 常见问题 (FAQ)

### Q: 为什么图片生成失败？
A: 可能缺少 matplotlib 和 numpy 依赖，运行：
```bash
pip install matplotlib numpy
```

### Q: 恒星颜色查询不可用？
A: 需要安装 pandas 依赖：
```bash
pip install pandas
```

### Q: 如何查看所有可用的中国传统色？
A: 数据太多无法一次显示，建议：
1. 使用 `-a` 按色系浏览
2. 查看 `color.json` 文件
3. 访问相关网站浏览

### Q: 自定义颜色存储在哪里？
A: 存储在 `data/custom_colors.json`，可手动编辑此文件。

### Q: RGB 查询找不到颜色？
A: 中国色库中的颜色是有限的，如果找不到，插件会显示该颜色的预览和转换后的 HEX 值。

### Q: 光谱型格式是什么？
A: 标准格式如 `O5V`、`G2V`、`M0III` 等，使用 `/color -t` 查看所有可用光谱型。
