# Dict 插件

综合词典插件，提供天文学专业术语的中英互译功能。

## 功能介绍

Dict 插件提供天文学专业术语的中英文互译功能，支持模糊查询和精确查询。

## 使用方法

### 基本命令

```
/dict <关键词>
/dict help
```

### 示例

```
/dict galaxy         # 查询 "galaxy" 的中文翻译
/dict 星系           # 查询 "星系" 的英文翻译
/dict help           # 显示帮助信息
```

## 配置说明

需要准备词典数据文件（TSV 格式）：

- `data/astrodict_en2cn.tsv` - 英译中词典
- `data/astrodict_cn2en.tsv` - 中译英词典

词典文件格式（制表符分隔）：
```
源词	目标词
galaxy	星系
```

## 功能特性

- 中英文双向查询
- 模糊匹配支持
- 结果缓存提高性能
- 支持多个匹配结果展示
- 自动判断查询语言

## 依赖

- pandas

## 注意事项

- 首次使用需确保词典文件存在
- 词典文件需使用 UTF-8 编码
- 查询结果最多显示前 10 条匹配
