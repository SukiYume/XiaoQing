# Echo 插件

示例插件，展示插件基本编写规范和命令处理流程。

## 功能介绍

Echo 是一个简单的示例插件，用于演示 XiaoQing 插件系统的基本用法和最佳实践。

## 使用方法

### 基本命令

```
/echo <文本>         # 回显任意文本内容
/hello              # 打招呼
```

### 示例

```
/echo 你好世界
/echo 这是一个测试消息
/hello
```

## 功能特性

- 简单的文本回显功能
- 友好的打招呼功能
- 展示基本的命令处理流程
- 标准的错误处理

## 适用场景

- 学习插件开发的入门示例
- 测试插件系统是否正常工作
- 作为新插件开发的模板

## 开发参考

这是一个标准的插件模板，包含：

1. 清晰的文档字符串
2. 标准的 `init()` 和 `handle()` 函数
3. 完善的异常处理
4. 格式化的帮助信息
5. 日志记录

新插件开发时可以参考此插件的代码结构。
