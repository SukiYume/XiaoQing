# Guess Number 插件

猜数字游戏插件，提供经典的猜数字小游戏。

## 功能介绍

一个简单有趣的猜数字游戏，系统会随机生成一个数字，用户通过多次猜测来找到正确答案。

## 使用方法

### 基本命令

```
/guess start [范围]     # 开始新游戏
/guess <数字>           # 猜测数字
/guess give_up          # 放弃当前游戏
/guess help             # 显示帮助信息
```

### 示例

```
/guess start            # 开始游戏（默认范围 1-100）
/guess start 1000       # 开始游戏（范围 1-1000）
/guess 50               # 猜测数字 50
/guess give_up          # 放弃游戏
/guess help             # 显示帮助
```

## 游戏规则

1. 使用 `/guess start [范围]` 开始游戏
2. 系统随机生成一个指定范围内的整数
3. 用户猜测数字，系统提示"太大"或"太小"
4. 继续猜测直到答对
5. 可以随时使用 `/guess give_up` 放弃游戏

## 功能特性

- 支持自定义数字范围
- 会话管理（每个用户独立游戏状态）
- 猜测次数统计
- 实时提示反馈
- 游戏状态持久化

## 适用场景

- 休闲娱乐
- 测试逻辑思维
- 群聊互动小游戏

## 注意事项

- 每个用户只能同时进行一个游戏
- 游戏状态会在机器人重启后丢失
- 猜测的数字必须在指定范围内
