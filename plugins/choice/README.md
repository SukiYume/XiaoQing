# Choice 随机选择插件

## 功能简介

本插件是一个智能随机选择助手，帮助你在多个选项中做出决定。无论是日常的吃喝玩乐选择，还是工作中的方案决策，都能快速给出随机结果。

## 主要特性

- 🎲 **随机选择**：从多个选项中随机选出一个或多个
- 🎯 **多选支持**：支持一次选择多个选项
- ⚖️ **加权选择**：通过重复选项实现概率加权
- 🔄 **去重模式**：可选择是否允许重复选择同一项
- 🎨 **友好展示**：使用多种表情符号美化结果
- 📊 **统计信息**：显示选项数量和选择结果

## 使用方法

### 基础用法

```
/choice 问题 选项1 选项2 选项3
/选择 吃啥 火锅 烤肉 披萨 日料
/决定 今天干啥 看电影 打游戏 学习
```

**输出示例：**
```
🎲 吃啥：**火锅**
```

### 多选模式

使用 `-n` 参数指定选择多个：

```
/choice 抽奖 小明 小红 小张 小李 -n 2
/选择 今天做什么 任务A 任务B 任务C 任务D -n 3
```

**输出示例：**
```
✨ 抽奖：
  1. **小红**
  2. **小李**

已从 4 个选项中选择 2 个
```

### 去重模式

使用 `-u` 或 `--unique` 参数确保不重复选择：

```
/choice 选课 数学 物理 化学 生物 -n 3 -u
```

这在选择人员、任务等不能重复的场景中很有用。

### 加权选择

通过重复选项来增加其被选中的概率：

```
/选择 今天心情 开心 开心 开心 难过 一般
# 开心出现3次，被选中概率更高
```

## 特殊功能

### 1. 自动调整选择数量

如果指定的选择数量超过选项总数，插件会自动调整为选项总数：

```
/选择 抽奖 A B C -n 5
# 实际只会选择3个（全选）
```

### 2. 多样的表情符号

每次选择会随机使用不同的表情符号，让结果更有趣：

🎲 🎯 ✨ 🌟 💫 🎰 🔮 🎪

### 3. 智能格式化

- 单选：简洁的一行显示
- 多选：清晰的列表展示
- 统计：显示选项和结果数量

## 技术细节

### 随机算法

- **单选/加权**：使用 `random.choice()` 或 `random.choices()`
- **多选（允许重复）**：使用 `random.choices()` 有放回抽样
- **多选（去重）**：使用 `random.sample()` 无放回抽样

### 参数解析

插件使用自定义的参数解析器，支持：
- 空格分隔的选项
- 特殊参数识别（`-n`, `-u`）
- 参数位置灵活（可以在任意位置）

### 日志记录

插件会记录：
- 问题内容（截取前20字符）
- 选项数量（包含去重后数量）
- 选择数量和模式
- 最终选择结果

## 常见问题 (FAQ)

### Q: 选择真的是随机的吗？
A: 是的，使用 Python 的 `random` 模块，基于伪随机数生成器，在统计上是均匀分布的。

### Q: 可以实现真正的加权选择吗？
A: 当前版本通过重复选项实现简单的加权。例如 `火锅 火锅 烤肉` 中火锅被选中的概率是 2/3。

### Q: 多选时会不会重复选到同一个？
A: 默认可能会重复，使用 `-u` 参数可确保每个选项最多被选择一次。

### Q: 选择数量可以大于选项数量吗？
A: 可以指定，但实际会调整为选项数量。例如 3 个选项选 5 个，实际只会选 3 个。

### Q: 可以用于抽奖吗？
A: 完全可以！使用 `/抽奖` 命令和 `-n` 参数指定中奖人数，使用 `-u` 确保不重复。

### Q: 如何让某个选项更容易被选中？
A: 在选项中多次输入该选项，例如：`/选择 吃啥 火锅 火锅 火锅 烤肉 披萨`
