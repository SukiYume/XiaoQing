# Earthquake 插件

地震快讯插件，从微博中国地震台网获取地震快讯。

## 功能介绍

实时监测中国地震台网的微博发布，自动推送 4 级及以上地震的快讯信息。

## 使用方法

### 基本命令

```
/earthquake          # 获取最新地震信息
/earthquake latest   # 获取最新地震信息
/earthquake help     # 显示帮助信息
```

### 示例

```
/earthquake          # 查看最新地震快讯
```

## 配置说明

无需额外配置，开箱即用。

插件会自动：
- 定期检查新的地震微博
- 过滤出 4 级及以上地震
- 记录已推送的地震信息避免重复

## 功能特性

- 实时监测中国地震台网微博
- 自动过滤 4 级以上地震
- 提取震级、震中位置、时间等信息
- 避免重复推送同一地震
- 支持图片展示

## 数据来源

- 中国地震台网微博（@中国地震台网速报）
- 微博 UID: 1904228041

## 注意事项

- 仅推送 4 级及以上地震
- 数据来自微博，可能存在延迟
- 需要网络连接获取微博数据
