# QingPet - QQ群宠物养成系统

## 功能概述

QingPet是一个在QQ群中运行的宠物养成插件，用户可以领养宠物、日常喂养互动、宠物会随时间变化并成长进化。支持社交玩法，如互访、送礼、组队活动、排行展示等。

## 主要功能

### 宠物系统
- **领养宠物**: 使用 `/宠物 领养 <名字>` 领养一只宠物
- **查看状态**: 使用 `/宠物 状态` 查看宠物详细信息
- **喂食**: 使用 `/宠物 喂食 [道具]` 喂食宠物
- **清洁**: 使用 `/宠物 清洁` 清洁宠物
- **玩耍**: 使用 `/宠物 玩耍` 和宠物玩耍
- **睡觉**: 使用 `/宠物 睡觉` 让宠物休息
- **训练**: 使用 `/宠物 训练` 训练宠物提升经验
- **探索**: 使用 `/宠物 探索` 探索获取奖励

### 进化系统
- 宠物会根据照顾质量进化成不同形态
- 经验值积累达到一定程度后触发进化
- 优秀的照顾可以获得更好的进化形态

### 经济系统
- **金币**: 主要货币，通过日常互动获得
- **商店**: 使用 `/宠物 商店` 查看可用道具
- **购买**: 使用 `/宠物 购买 <道具> [数量]` 购买道具
- **背包**: 使用 `/宠物 背包` 查看拥有的道具

### 社交功能
- **互访**: 使用 `/宠物 互访 @QQ号` 访问他人宠物
- **送礼**: 使用 `/宠物 送礼 @QQ号 <道具> [数量]` 送出礼物
- **排行榜**: 使用 `/宠物 排行 [类型]` 查看各种排行榜
- **每日任务**: 使用 `/宠物 任务` 查看每日任务

### 管理功能
- **开启/关闭**: 使用 `/宠物 管理 开启/关闭` 控制插件状态
- **配置管理**: 使用 `/宠物 管理 配置 查看/设置` 管理群配置
- **用户管理**: 使用 `/宠物 管理 重置/封禁/解封 @QQ号` 管理用户
- **操作日志**: 使用 `/宠物 管理 日志 [数量]` 查看操作记录

## 定时任务

插件包含以下定时任务：
- **状态衰减**: 每分钟执行，宠物状态值会随时间自然衰减
- **每日重置**: 每天00:00执行，重置用户的每日计数器
- **每周活动**: 每周一10:00执行，进行每周活动结算

## 数据存储

插件使用SQLite数据库存储数据，数据库文件位于 `plugins/qingpet/data/qingpet.db`

## 配置项

### 群级配置
- `enabled`: 插件是否启用
- `economy_multiplier`: 经济倍率
- `decay_multiplier`: 衰减倍率
- `trade_enabled`: 是否开启交易功能
- `natural_trigger_enabled`: 是否开启自然触发
- `activity_enabled`: 是否开启活动功能

## 注意事项

1. 每个用户在每个群中只能养一只宠物
2. 宠物状态会随时间自然衰减，需要定期照顾
3. 各项操作都有冷却时间，避免频繁操作
4. 每日金币获取和操作次数有限制
5. 管理员可以对违规用户进行封禁处理

## 扩展开发

如需扩展功能，可以参考以下模块：
- `models/`: 数据模型定义
- `services/`: 业务逻辑服务
- `commands/`: 命令处理函数
- `utils/`: 工具函数和常量定义
