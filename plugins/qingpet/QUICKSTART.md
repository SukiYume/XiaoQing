# QingPet 快速开始指南

## 安装

QingPet插件已经集成在项目中，无需额外安装。确保插件目录结构完整即可。

## 启用插件

1. 在QQ群中，使用管理员账号发送：
   ```
   /宠物 管理 开启
   ```

2. 确认插件已启用：
   ```
   /宠物 管理 配置 查看
   ```

## 基础使用

### 1. 领养宠物
```
/宠物 领养 小白
```
这会给你一颗宠物蛋。

### 2. 查看状态
```
/宠物 状态
```
可以看到宠物的各项属性值。

### 3. 喂食宠物
```
/宠物 喂食 apple
```
喂食可以提升饥饿和心情。

### 4. 清洁宠物
```
/宠物 清洁
```
清洁可以提升清洁和健康值。

### 5. 和宠物玩耍
```
/宠物 玩耍
```
玩耍可以提升心情和亲密度。

## 进化系统

宠物会根据照顾质量进化：
1. **蛋 → 幼年**: 获得10经验后自动破壳
2. **幼年 → 成长期**: 达到50经验后进化，照顾质量影响形态
3. **成长期 → 成熟期**: 达到100经验后进化
4. **成熟期 → 老年**: 年龄增长后自然老化

照顾评分 = (饥饿+心情+清洁+精力+健康)/5

## 注意事项

1. **冷却时间**: 各项操作都有冷却时间，请耐心等待
2. **每日限制**: 金币获取和操作次数有每日上限
3. **状态衰减**: 宠物状态会随时间自然衰减，需要定期照顾
4. **生病机制**: 健康值过低会导致生病，需要治疗
5. **睡觉机制**: 睡觉时无法互动，但会恢复精力

## 常见问题

**Q: 如何让宠物进化？**
A: 通过喂食、玩耍、训练等操作积累经验，经验达到要求后自动进化。

**Q: 宠物生病了怎么办？**
A: 使用 `/宠物 治疗 medicine` 或购买药品治疗。

**Q: 为什么操作失败？**
A: 可能是冷却时间未到、精力不足、或今日次数已达上限。

**Q: 如何获得更多金币？**
A: 通过日常互动、访问他人宠物、完成每日任务等方式获得。

**Q: 可以养多只宠物吗？**
A: 每个用户在每个群中只能养一只宠物。

**Q: 宠物会死亡吗？**
A: 不会死亡，但健康值过低会生病，照顾不佳可能离家旅行。
