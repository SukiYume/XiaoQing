# Memo 插件

备忘录插件，提供简单的笔记和提醒功能。

## 功能介绍

Memo 插件允许用户创建、查看、删除个人备忘录，支持多用户独立的备忘录管理。

## 使用方法

### 基本命令

```
/memo add <内容>        # 添加新备忘录
/memo list              # 列出所有备忘录
/memo del <编号>        # 删除指定备忘录
/memo clear             # 清空所有备忘录
/memo help              # 显示帮助信息
```

### 示例

```
/memo add 明天下午3点开会
/memo add 买书：《天体物理学》
/memo list
/memo del 1
/memo clear
/memo help
```

## 功能特性

- 多用户支持（每个用户独立的备忘录）
- 持久化存储（重启后数据不丢失）
- 编号管理（方便删除指定条目）
- 批量清空功能
- 时间戳记录

## 数据存储

备忘录数据存储在 `data/memo.json` 文件中，格式如下：

```json
{
  "user_id": [
    {
      "id": 1,
      "content": "备忘内容",
      "timestamp": "2026-02-04 12:00:00"
    }
  ]
}
```

## 注意事项

- 每个用户的备忘录相互独立
- 删除操作不可恢复，请谨慎使用
- 备忘录数量建议不超过 100 条以保持性能
- 支持中文和 emoji
