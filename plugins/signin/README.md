# Signin 插件

签到插件，提供每日签到和积分管理功能。

## 功能介绍

Signin 插件允许用户每日签到，累积积分和连续签到天数，支持签到统计和排行榜功能。

## 使用方法

### 基本命令

```
/signin                 # 每日签到
/signin info            # 查看签到信息
/signin rank            # 查看签到排行榜
/signin help            # 显示帮助信息
```

### 示例

```
/signin                 # 今日签到
/signin info            # 查看我的签到信息
/signin rank            # 查看排行榜
/signin help            # 显示帮助
```

## 功能特性

- 每日签到（每天只能签到一次）
- 连续签到天数统计
- 积分累积系统
- 签到排行榜
- 多用户独立数据
- 数据持久化存储

## 签到规则

- 首次签到奖励: 10 积分
- 每日签到奖励: 5 积分
- 连续签到额外奖励: 1 积分/天
- 断签后连续天数清零

## 数据存储

签到数据存储在 `data/signin.json` 文件中：

```json
{
  "user_id": {
    "last_signin_date": "2026-02-04",
    "total_days": 30,
    "continuous_days": 7,
    "points": 165,
    "last_points": 12
  }
}
```

## 排行榜

排行榜按以下维度展示前 10 名用户：
1. 总签到天数
2. 连续签到天数
3. 累积积分

## 注意事项

- 每天只能签到一次（按日期判断）
- 连续签到需要每天都签到，否则清零
- 积分暂无兑换功能，仅作统计
- 数据会永久保存
